# Local URL Shortener

Сервис для локального сокращения ссылок, написанный на Go

## Возможности

* Создание коротких ссылок через HTTP API
* Редирект на оригинальный URL
* Хранение данных в памяти (in-memory)
* Наличие тестов

---

## Запуск

### Требования

* Go 1.21+

### Запуск локально

go run .
По умолчанию сервер стартует на:

http://localhost:8080
---

## API

### Создать короткую ссылку

POST /api/shorten

Пример запроса:

{
  "url": "https://example.com",
  "ttl_seconds": 3600
}
* url — исходная ссылка
* ttl_seconds — время жизни в секундах (необязательно)

Пример ответа:

{
  "code": "abc12345",
  "short_url": "http://localhost:8080/abc12345",
  "original_url": "https://example.com",
  "created_at": "2026-04-25T12:00:00Z",
  "expires_at": "2026-04-25T13:00:00Z"
}
---

### Редирект

GET /{code}

Пример:

http://localhost:8080/abc12345
Сервис выполнит редирект на исходный URL.

---

### Проверка состояния сервиса

GET /healthz
Ответ:

{
  "status": "ok"
}
---
